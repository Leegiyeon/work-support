work-support에 경력 자산화 기능을 구현한다.

목표:
프로젝트, 업무 로그, 개선 성과 데이터를 기반으로 이력서/포트폴리오/면접 답변에 활용할 수 있는 문장을 생성하고 저장할 수 있어야 한다.

이번 범위:
1. 프로젝트별 경력 자산 화면
2. 업무 로그 기반 수행 업무 요약
3. 개선 성과 기반 성과 요약
4. 이력서용 문장 생성
5. 경력기술서용 문장 생성
6. 포트폴리오용 프로젝트 설명 생성
7. 면접 답변용 STAR 구조 생성
8. 생성 결과 저장
9. 사용자가 직접 수정 가능
10. Markdown 복사 기능

중요:
- 실제 work_logs와 project_outcomes를 근거로 사용한다.
- 수치가 없으면 임의로 만들지 않는다.
- 과장된 표현보다 실제 기여와 개선 내용을 중심으로 작성한다.
- AI API가 아직 연결되지 않았다면, 우선 템플릿 기반 생성으로 구현하고 추후 AI 연동 가능하게 구조화한다.

완료 기준:
- 프로젝트별로 이력서에 쓸 수 있는 문장 초안을 확인할 수 있어야 한다.
- 생성된 문장을 수정하고 저장할 수 있어야 한다.
